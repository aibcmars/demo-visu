# mxgraph-js-playground

Tested with node 
- node v12.14.1
- npm 6.13.4

## Run
```
npm install
npm run dev
```

Visit `http://localhost:5005/` to take a look.


## Development

Use [mxgraph-js](https://github.com/jgraph/mxgraph-js) lib with typescript support
* https://github.com/asual/mxgraph-factory
* https://github.com/lgleim/mxgraph-typings (is used?)
